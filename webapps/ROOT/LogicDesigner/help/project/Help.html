Project

A Project represents the grouping of your database location, and your logic / security settings. It is what creates the RESTful server.

  • Your account can include many projects

  • They are all running, but you manage the contents of only the current project at one time, using the Logic Designer
Use the drop down at the top to

  • Select your current project

  • Create new ones with by connecting to a database, or

  • Import an exported project

Project include settings such as

  • URL fragment which is used to construct your API.

  • Libraries contain reusable code that you can use in JavaScript events (e.g., date arithmetic, database integration, extended logic)

  • Authentication providers that integrate Espresso with your enterprise security (lists of valid users)
